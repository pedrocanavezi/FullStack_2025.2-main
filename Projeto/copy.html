Introdução

C++ (em português: "cê mais mais", em inglês "see plus plus") é uma linguagem compilada multi-paradigma e de uso geral. Desde os anos 1990 é uma das linguagens comerciais mais populares, usada também na academia pelo grande desempenho.

Java é uma linguagem orientada a objetos criada na década de 90 pela Sun Microsystems. Em 2008 foi adquirida pela Oracle Corporation.

Java tem estrutura semelhante à de C e compartilha com C++ a orientação a objetos e um alto grau de semelhança.
